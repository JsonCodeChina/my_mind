## 💻 安装和配置

### 系统要求
- **Node.js**：18 或更新版本
- **账户**：Claude.ai 或 Claude Console 账户
- **操作系统**：支持主流操作系统

### 安装命令
```bash
# 全局安装
npm install -g @anthropic-ai/claude-code

# 启动
claude
```

### 配置文件
- `.claude/` - 项目级配置目录
- `commands/` - 自定义命令
- 其他配置文件根据需要创建
